# Pratyaksh.dev

Personal site and blog hosted with GitHub Pages.

## Add a Blog Post

Create a Markdown file in `_posts/` using this filename format:

```text
YYYY-MM-DD-post-title.md
```

Use front matter at the top:

```markdown
---
title: "Post Title"
description: "One short sentence shown on the blog index."
---

Write the post in Markdown here.
```

Commit and push the file. GitHub Pages will publish it at:

```text
/blog/post-title/
```

## Current Structure

- `index.html`: home page
- `blog/index.html`: list of posts
- `_posts/`: Markdown blog posts
- `_layouts/`: shared page templates
- `assets/styles.css`: site styles
